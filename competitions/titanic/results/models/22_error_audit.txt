SECTION 8: WHAT'S LEFT — THE IRREDUCIBLE RESIDUAL
=================================================================

The ~94 test errors likely come from:

1. MALE PCLASS 3 — men in steerage who died (model already predicts this)
   But some survived. These are exceptions to the dominant pattern.
   The model has almost no features to identify which Pc3 men survived.

2. FEMALE PCLASS 3 — the most contested subgroup
   ~50% survival rate means any prediction is close to a coin flip.
   Aggressive correction hurts Kaggle (confirmed 3 times).
   The safe move is to predict 'survived' and accept ~50% errors here.

3. MALE PCLASS 1 — wealthy men, ~37% survival
   The model correctly saves some via ticket rules and high-status features.
   But many Pc1 men died despite privilege. These are hard to predict.

AUDIT SUMMARY
=================================================================

1. The model's ~94 test errors are concentrated in three groups:
   - male Pclass 3 survivors (unpredictable from available features)
   - female Pclass 3 deaths (coin-flip subgroup, correction hurts)
   - male Pclass 1 deaths (wealthy men who chose not to board lifeboats)

2. The model adds value over gender-only primarily by:
   - correctly identifying some male survivors via class/family features
   - correctly identifying some female Pclass 3 deaths via fare/family features
   - ticket-rule overrides for passengers with strong group signal

3. The remaining errors are largely irreducible with available features:
   - individual survival decisions (heroism, panic, luck) aren't in the data
   - the ~50% female Pclass 3 survival rate creates a structural error floor
   - most male Pclass 3 deaths are correctly predicted, but the rare survivors
     have no distinguishing features

4. The honest ceiling estimate:
   - gender-only: ~0.7655 (320/418)
   - current best: ~0.7751 (324/418)
   - theoretical with perfect group rules: ~0.79-0.80 (330-335/418)
   - beyond that requires features not in this dataset
